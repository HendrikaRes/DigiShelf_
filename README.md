📚 Tentang Project

DigiShelf merupakan aplikasi sistem informasi perpustakaan berbasis web yang dibuat untuk mempermudah proses pengelolaan data perpustakaan secara digital.
Aplikasi ini memiliki fitur manajemen anggota, pengelolaan buku, peminjaman, dan pengembalian buku dengan tampilan modern dan responsif.

Project ini dikembangkan menggunakan framework Laravel 11 dengan antarmuka yang sederhana namun mudah digunakan.
